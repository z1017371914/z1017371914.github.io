# Java技术的趋势

- 模块化。在应用系统不断复杂、越来越庞大的时候，模块化是一个重要的应用方向。

- 混合语言。有越来越多基于Java虚拟机的语言诞生，他们可能在应用中的不同层次发挥作用。

- 多核并行

- 丰富语法

- 64位虚拟机
